# simple-dragdrop
このリポジトリは、**標準の Tkinter** と **[tkinterdnd2](https://pypi.org/project/tkinterdnd2/)** を使って
「ファイルをウィンドウにドラッグ＆ドロップすると中身を表示する」だけの、最小限のサンプルアプリです。

---

## 動作確認環境

- Python 3.10+（3.8 以降ならだいたい動きます）
- macOS / Windows で確認
- `tkinter` が動く環境であること
- `tkinterdnd2` を pip インストール済みであること

※ Linux でも Tk が入っていればほぼ同じように動きます。

## インストール

```bash
git clone https://github.com/yourname/tkinter-dnd-sample.git
cd tkinter-dnd-sample
pip install -r requirements.txt
```
